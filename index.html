연구 주제를 선정하게 된 이유
코로나19를 직접 경험하며 뉴스에서 매일 접하던 확진자 그래프와 재생산지수(R값)가 실제로 어떤 수학적 원리에 기반하는지 궁금증을 가지게 되었다. 단순히 그래프가 증가하거나 감소하는 것을 넘어, 그 변화율—즉 미분—이 감염병의 심각성을 나타내는 핵심 지표임을 알게 되었고, 고등학교 미적분에서 배운 지수·로그함수의 미분이 이러한 현상을 수학적으로 기술하는 직접적인 도구가 된다는 점에 흥미를 느껴 탐구 주제로 선정하였다.
관련 교과 단원
미적분 — 지수함수의 미분로그함수의 미분자연상수 e함수의 증가·감소
본 론
핵심 개념
(e^x)' = e^x
(ln x)' = 1/x
(a^x)' = a^x · ln a

자연상수 e란 극한 lim(1 + 1/n)^n (n→∞) ≈ 2.718으로 정의되며, 지수함수 ex는 자기 자신을 미분해도 변하지 않는 유일한 함수이다. 이 성질이 연속적으로 변하는 자연 현상을 모델링하는 데 핵심적으로 활용된다.
진로 관련 개념
데이터 시각화로그 스케일 그래프시뮬레이션 알고리즘

개발자 관점에서, 감염병 시뮬레이션 프로그램은 미분방정식을 수치적으로 계산(오일러 방법 등)하는 알고리즘으로 구현된다. 또한 확진자 수가 수십~수백만에 이를 때 로그 스케일로 그래프를 표현하면 변화율의 추이를 명확히 시각화할 수 있는데, 이것이 (ln x)' = 1/x의 직접적인 응용이다.
내용 요약
① SIR 모델이란?
감염병 확산을 수학적으로 기술하는 가장 기본적인 모델로, 전체 인구를 세 집단으로 나눈다.
S (Susceptible) 감염 가능 집단   I (Infectious) 감염자 집단   R (Recovered) 회복(면역) 집단
② 핵심 수식
감염자 수 I(t)는 초기에 지수함수적으로 증가한다:
I(t) ≈ I₀ · e^((β-γ)t)
여기서 β는 감염률, γ는 회복률이다. 이를 로그 변환하면:
ln I(t) = ln I₀ + (β-γ)t
즉, 로그 스케일에서 감염자 곡선의 기울기 = (β−γ)이며, 이것이 곧 미분값(변화율)이다.
③ 기초감염재생산지수 R₀
R₀ = β / γ
R₀ > 1이면 I(t)의 미분값이 양수 → 감염 확산
R₀ < 1이면 I(t)의 미분값이 음수 → 감염 소멸

즉, 뉴스에서 말하는 "R값이 1을 넘었다"는 것은 수학적으로 감염자 함수의 도함수가 양수임을 의미한다.
시뮬레이션: SIR 모델 감염자 곡선 (R₀ = 2.5 vs R₀ = 0.8)
알게 된 점 & 결론
지수함수 ex를 미분하면 자기 자신이 된다는 성질이 단순한 수학 공식이 아니라, 자연에서 "변화율이 현재 상태에 비례하는" 모든 현상을 기술하는 보편적 언어임을 깨달았다. 감염병 확산뿐 아니라 인구 증가, 방사성 붕괴, 복리 계산 등에도 동일하게 적용된다.
로그함수의 미분 (ln x)' = 1/x는 데이터를 로그 스케일로 변환할 때 기울기가 곧 변화율(성장률)을 나타내게 해주며, 이것이 데이터 분석과 시각화에서 로그 스케일이 자주 사용되는 수학적 이유임을 이해했다.
R₀ (기초감염재생산지수)가 1을 기준으로 감염 확산 여부가 결정된다는 사실은, 수학적으로 도함수의 부호 판정 문제와 정확히 동일하다. 미적분의 "증가·감소 판정"이 공중보건 정책 결정에 직접 활용된다는 점이 인상적이었다.
개발자로서, 이러한 미분방정식을 컴퓨터로 수치 계산하는 오일러(Euler) 방법이나 룽게-쿠타(Runge-Kutta) 알고리즘이 실제 시뮬레이션 소프트웨어의 핵심임을 알게 되었고, 수학과 프로그래밍이 긴밀하게 연결되어 있음을 체감했다.
후속 활동 계획 및 다짐
이번 탐구를 통해 미적분의 개념이 현실 세계와 이렇게 직접적으로 연결되어 있다는 것을 처음으로 실감하였다. 앞으로는 SIR 모델을 실제로 Python 또는 JavaScript로 구현하여 R₀ 값에 따른 감염 곡선 변화를 시각화하는 프로그램을 만들어보고 싶다. 또한 지수·로그 미분이 활용되는 다른 분야(머신러닝의 손실함수, 정보이론의 엔트로피 등)로 탐구를 확장하여 수학과 개발이 연결되는 지점을 더 깊이 이해하고자 한다.
