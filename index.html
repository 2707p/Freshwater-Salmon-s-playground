1. 시뮬레이터 핵심 로직 (학술적 근거)

경제학에서 가격 변화에 따른 수요 변화를 계산하는 공식은 다음과 같습니다.

Ed = %ΔQ / %ΔP
  • 가격탄력성(Ed) 도출: 예시(세율 10% 상승 시 소비 5% 감소)를 공식에 대입하면 탄력성은 -0.5가 됩니다. (실제 국내외 가당음료(SSB) 연구에서 설탕세 탄력성은 보통 -0.5 ~ -1.2 사이로 측정되므로, 매우 현실적이고 방어하기 좋은 수치입니다.)

3. 파이썬(Python) 시뮬레이터 코드

데이터 분석 파트가 포함된 보고서라면 아래 코드를 첨부하거나 실행하여 결과값을 추출할 수 있습니다.

def sugar_tax_simulator(tax_rate_pct, beverage_price, elasticity=-0.5, market_size=105.26):
    '''설탕세율과 음료 가격을 입력받아 정책 효과를 시뮬레이션하는 함수'''
    
    # 1. 예상 소비 감소율(%) 계산
    consumption_decrease_pct = abs(tax_rate_pct * elasticity)
    
    # 2. 세금 부과 후 잔존하는 소비 비율
    remaining_consumption_ratio = 1 - (consumption_decrease_pct / 100)
    
    # 3. 예상 세수 계산 (시장규모 * 세율 * 잔존소비비율)
    expected_revenue = market_size * (tax_rate_pct / 100) * remaining_consumption_ratio
    
    print(f"--- 설탕세 정책 시뮬레이션 결과 ---")
    print(f"설정 세율: {tax_rate_pct}% (음료 평균가: {beverage_price}원 기준)")
    print(f"▶ 예상 소비 감소량: {consumption_decrease_pct:.1f}%")
    print(f"▶ 예상 확보 세수: 약 {expected_revenue:.1f}억 원")

# 실행 예시
sugar_tax_simulator(tax_rate_pct=10, beverage_price=2000)
